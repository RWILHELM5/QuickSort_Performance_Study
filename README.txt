## How to Compile
Use the Makefile to compile the programs:
1. To compile the QuickSort program:
    make
This generates an executable named Wilhelm_Ryan_QuickSort.

2. To compile the input file generator:
    g++ -o InputFileGenerator Wilhelm_Ryan_InputFileGenerator.cpp
